# patterns/

Vacío a propósito.

Acá vivirían los patrones detectados entre entrevistas ("el 80% de los talleres mencionan el mismo problema de turnos", "todos los restaurantes usan WhatsApp como canal principal") — el insumo real para las primeras Capabilities oficiales.

No hay ningún proceso automático que llene esta carpeta todavía, y no debería haberlo. Eso es, literalmente, el Knowledge Engine (`inteliar-spec/FOUNDING_DECISIONS.md`, hipótesis 🟡), y requiere volumen real de organizaciones que hoy no existe — con 20 o 100 entrevistas no hay suficiente volumen para que una IA aprenda patrones reales por sí sola.

**Cómo se llena, por ahora:** a mano. Después de tener 15-20 entrevistas en una misma categoría de `../industries/`, alguien (una persona) las lee y escribe acá lo que nota en común. Cuando ese proceso manual se vuelva repetitivo y obvio, ahí se justifica automatizarlo — no antes.
